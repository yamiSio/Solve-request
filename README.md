# 解决跨域问题的方法(Vue项目用node代理)
## 主要是由于浏览器的同源策略造成的！
## 在登录页面，获取图片验证码
## 1.首先在utils文件内的api.js中进行发送请求设置  
```
//先设置一个前置路径  
let base='';  
//传送json格式的post请求  
export contst postRequest=(url,params)=>{  
  return axios({  
    method:'post',  
    url:` ${base}${url} `,  
    data:params
  })  
}  
```

## 2.在登录页面组件中获取图片验证码
```
//需要引入组件
```
